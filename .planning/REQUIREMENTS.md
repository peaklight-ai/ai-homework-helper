# Requirements

**Project:** Valid - AI Homework Helper
**Version:** v1 (MVP)
**Last Updated:** 2026-01-26

## v1 Requirements

### Bug Fixes

- [ ] **BUG-01**: Fix app blocking after correct answer — AI should entice user to try another question, not follow up on the solved problem
- [ ] **BUG-02**: Fix email configuration — emails going to junk folder instead of inbox

### Class Management

- [ ] **CLASS-01**: Teacher can bulk import students via CSV upload
- [ ] **CLASS-02**: Students organized by class (not mixed student lists)
- [ ] **CLASS-03**: Teacher can create, edit, and delete classes

### Teacher Settings

- [ ] **SETTINGS-01**: Teacher can configure topic and difficulty level per student
- [ ] **SETTINGS-02**: Teacher can upload hints/strategies linked to specific exercises

### Homework & Exercises

- [ ] **HW-01**: Teacher can upload homework by grade, class, or individual student
- [ ] **HW-02**: Teacher can assign exercises to specific students

### Diagnostic Assessment

- [ ] **DIAG-01**: Diagnostic test administered automatically on student's first login
- [ ] **DIAG-02**: Diagnostic results auto-set student's initial difficulty level
- [ ] **DIAG-03**: Diagnostic results sent to teacher for visibility

### Student Experience

- [ ] **STUDENT-01**: Student can see their targets/objectives (set by teacher)
- [ ] **STUDENT-02**: Dynamic strengths/weaknesses profile per student (tracks mastery by topic)
- [ ] **STUDENT-03**: Teacher can view cognitive reasoning patterns — how student thinks through problems, not just right/wrong

---

## v2 Requirements (Deferred)

- Student uploads own exercises (teacher-only upload for MVP)
- Real-time chat between teacher and student
- Arabic reading comprehension
- French language version
- ClassDojo integration
- Visual learning (diagrams, maps)

---

## Out of Scope

| Exclusion | Reasoning |
|-----------|-----------|
| Student-uploaded exercises | Reduces complexity; teacher controls content for MVP |
| Teacher-student chat | Not essential for tutoring workflow |
| Multi-language support | Post-MVP internationalization |
| External integrations | Focus on core product first |

---

## Validated (Existing Functionality)

- ✓ Teacher can sign up and log in with email/password
- ✓ Teacher can add individual students with login codes
- ✓ Student can log in with 6-character code
- ✓ Student receives AI-guided Socratic tutoring
- ✓ AI streams responses in real-time
- ✓ Student can upload homework photos for OCR extraction
- ✓ XP points system tracks student progress
- ✓ Teacher can view student list and basic progress
- ✓ Topic/domain selection available (partial)
- ✓ 30+ sample math problems across domains
